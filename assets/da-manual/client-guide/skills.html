The Skills Display


By clicking on the skills icon you can see what skills you have learned. You can also use these skills on items or other Aislings. The main window will change to show you the skills currently available to you.




Learning skills
As you travel in Temuair, you will be able to learn skills from the mundanes in various towns. If you travel to the different cities and visit the masters of the various classes (for instance, if you are a wizard, visiting Dar in Mileth), you will find that some will teach you new skills...for a price.

Using skills
If the skill affects an Aisling, you can use it in one of two ways: by double clicking on the skill icon, and then clicking on the Aisling you wish to cast it on; or by dragging the skill icon onto the Aisling you wish to cast it on.

Getting rid of a skill
If you ever desire to forget a skill you have learned, visit any master of the arts anywhere in Temuair (Dar, for example). All of the masters will help you forget a skill that you no longer want knowledge of.
